# 实盘策略说明

## 功能概览
- 新增「实盘策略」页面，管理已配置策略
- 每天 10:00 自动评估策略信号
- 触发买卖点时发送邮件通知
- 回测页面一键添加到实盘策略

## 邮件配置（.env）
```
EMAIL_HOST=smtp.example.com
EMAIL_PORT=587
EMAIL_USER=your_user
EMAIL_PASSWORD=your_password
EMAIL_FROM=your_user@example.com
EMAIL_TO=receiver1@example.com,receiver2@example.com
EMAIL_USE_TLS=true
```

## 策略支持
- 定投（dca）
- 智能定投（smart_dca）
- 均值回归（mean_reversion）
- 多轮均值回归（mean_reversion_multi）
- 趋势跟随（trend_following）

## 自动运行规则
- 每天 10:00 运行一次
- 若触发买卖点，发送邮件并记录最近信号

## 回测一键添加
在「策略回测」中运行回测后，点击「添加到实盘策略」即可生成同配置的实盘策略。
